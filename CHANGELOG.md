# Changelog

## [0.1.0] - 2026-04-23

### Added
- Initial release.
- Diagnostics for `SKILL.md` files: frontmatter presence, required fields, name format, description length, secret leaks, unknown fields, empty body.
- Self-contained TypeScript validator (no Python runtime needed).
- `claudeSkillCheck.enable` and `claudeSkillCheck.fileGlob` settings.
- `Claude Skill Check: Lint Active File` command.
